# CareerNest

CareerNest is a full-stack MERN job portal where recruiters can post jobs and candidates can apply.

## Features

* JWT Authentication
* Role-based Access (Recruiter / Candidate)
* Recruiter Dashboard
* Job Posting System
* Candidate Application Tracking
* Resume Upload
* Profile Picture Upload
* Cloudinary Integration
* Protected Routes
* Responsive UI

## Tech Stack

Frontend:

* React.js
* Tailwind CSS
* Axios

Backend:

* Node.js
* Express.js
* MongoDB
* JWT
* Multer
* Cloudinary

## Installation

Clone repository:

```bash
git clone <your-github-link>
```

Install dependencies:

```bash
npm install
cd client
npm install
```

Run backend:

```bash
node server.js
```

Run frontend:

```bash
npm run dev
```
